# Portfolio Site

A responsive personal portfolio site for Rene Mayen, built with HTML and CSS. It highlights software engineering projects, technical skills, and contact links in a clean recruiter-friendly layout.

## Features

- Responsive layout for mobile and desktop
- Project cards with direct GitHub links
- Skills section focused on systems, backend, and web development
- Contact section with email and LinkedIn links
- No build step required

## Run Locally

Open `index.html` in a browser.

For a local development server with Node.js:

```bash
npx serve .
```

You can also use Python if installed:

```bash
python -m http.server 8080
```

## Project Structure

```text
portfolio-site/
  index.html
  styles.css
```

## Customization

- Update project cards in `index.html`
- Replace contact links with your preferred email, LinkedIn, and GitHub links
- Adjust colors in `styles.css` under the `:root` variables

## Deployment

This site can be deployed with GitHub Pages:

1. Push the repository to GitHub.
2. Open repository settings.
3. Go to Pages.
4. Deploy from the `main` branch.
